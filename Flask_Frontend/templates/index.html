Welcome to our "JEERS UFC" Project

ABOUT OUR PROJECT

The JEERS group selected to work with the UFC Fighter Data. The Dataset encompasses numerous fighter statistics per match, per player. The reason the group chose this dataset is not only because the group members are UFC enthusiasts, but they also found the dataset appealing as it contains major numerical values. The JEERS group believes there lies plenty of functionality within the dataset and is eager to drive some of these values through machine learning methods to obtain solutions that can answer business questions.

THE GOAL

The group will use an all time UFC fight history dataset and machine learning to help determine or predict a future winner in UFC match ups. Using machine learning algorithms, specifically using Random Forest, can a fight winner be determined based on fighter demographics: weight, weightclass, height, age, reach, stance, previous fight record, fight style and fight statistics. The dataset was obtained from Kaggle.com.
